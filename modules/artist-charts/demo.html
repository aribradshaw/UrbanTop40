Usage Instructions

WordPress Shortcode

[artist_charts artist="the beatles" width="100%" height="600px"]

Parameters

  • artist - Artist name (currently only "the beatles" is supported)
  • width - Chart width (default: "100%")
  • height - Chart height (default: "600px")
Note: This demo requires Chart.js to be loaded. In a WordPress environment, you would need to enqueue Chart.js or use a plugin that provides it.

Features

  • Interactive line chart showing Billboard chart positions over time
  • Zoom functionality with mouse wheel
  • Responsive design for mobile and desktop
  • Song legend with peak positions and chart weeks
  • Modern dark theme with smooth animations
  • AJAX-powered data loading
  • Error handling and retry functionality
