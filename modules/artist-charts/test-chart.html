How It Works

  1. Data Loading: The module reads TypeScript files from assets/artistcharts/
  2. Chart Rendering: Creates horizontally-scrollable charts with position 1 at top, 100 at bottom
  3. Interactive Features: Hover tooltips, click events, and responsive design
  4. WordPress Integration: Uses shortcodes for easy embedding in posts and pages

File Structure

The module expects this file structure:

modules/
├── artist-charts/
│   ├── artist-charts.php      # WordPress integration
│   ├── artist-charts.css      # Styling
│   ├── artist-charts.js       # Chart functionality
│   └── README.md             # Documentation
assets/
└── artistcharts/
    └── the_beatles_data.ts   # Artist data file
