Basic Usage:

[artist_chart artist="the_beatles_data"]

With Custom Options:

[artist_chart artist="the_beatles_data" height="500" width="100%" chart_type="line"]
Important: Use the exact filename (without .ts extension) as the artist parameter. For example, if your file is named "the_beatles_data.ts", use "the_beatles_data" as the artist parameter.

Available Parameters:

  • artist - The artist identifier (required)
  • height - Chart height in pixels (default: 400)
  • width - Chart width (default: 100%)
  • chart_type - Chart type: line, area, or bar (default: line)
